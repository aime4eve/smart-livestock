Subscription 事件

S5	SubscriptionCancelledEvent	cancel()	订阅取消	通知: 告知 owner
S6	SubscriptionExpiredEvent	markExpired()	订阅到期	Ranch/IoT: 限制功能访问
Contract 事件
#	事件名	触发点	含义	跨上下文消费方
C1	ContractCreatedEvent	create()	合同草稿创建	无（内部审计）

C3	ContractTerminatedEvent	terminate()	合同终止	通知: 告知合作方
C4	ContractExpiredEvent	markExpired()	合同到期	通知: 告知合作方

#	事件名	触发点	含义	跨上下文消费方

R2	RevenueSettledEvent	settle()	结算完成	通知: 结算通知
SubscriptionService 事件
#	事件名	触发点	含义	跨上下文消费方
L1	ServiceActivatedEvent	activate()	Licensed 服务激活	通知: 服务已上线
