## 燃气表

### 1. 设置燃气单价 (gasUnitPrice)

```json
{
    "method": "gasUnitPrice",
    "params": {
        "gasUnitPrice": 3.5
    }
}
```

### 2. 燃气充值 (gasCharge)

```json
{
    "method": "gasCharge",
    "params": {
        "gasBalance": 200
    }
}
```

### 3. 设置燃气用量 (gasUsage)

```json
{
    "method": "gasUsage",
    "params": {
        "gasUsage": 50.5
    }
}
```

### 4. 设置燃气余量 (gasSurplus)

```json
{
    "method": "gasSurplus",
    "params": {
        "gasSurplus": 150.5
    }
}
```

### 5. 设置燃气余额 (gasBalance)

```json
{
    "method": "gasBalance",
    "params": {
        "gasBalance": 200
    }
}
```

### 6. 阀门控制 (valveControl)

```json
{
    "method": "valveControl",
    "params": {
        "valveStatus": 0
    }
}
```

| valveStatus | 说明 |
|---------|------|
| 0 | 关阀 |
| 1 | 开阀 |

## 暖气控制阀

### 1. 设置目标温度 (SetTargetTemperature)

```json
{
    "method": "SetTargetTemperature",
    "params": {
        "targetTemperature": 25.5,
        "temperatureTolerance": 0.5
    }
}
```

### 2. 设置阀门开度 (SetValveOpening)

```json
{
    "method": "SetValveOpening",
    "params": {
        "valveOpening": 50
    }
}
```

| valveOpening | 说明 |
|--------------|------|
| 0-100 | 阀门开度百分比 |

## 超声波水表

### 1. 阀门控制 (ValveControl)

```json
{
    "method": "ValveControl",
    "params": {
        "valveStatus": 0
    }
}
```

| valveStatus | 说明 |
|-------------|------|
| 0 | 开阀 |
| 1 | 关阀 |
| 2 | 疏通 |

### 2. 设置计量模式 (SetMeteringMode)

```json
{
    "method": "SetMeteringMode",
    "params": {
        "meteringMode": 0
    }
}
```

| meteringMode | 说明 |
|--------------|------|
| 0 | 双脉冲 |
| 1 | 单脉冲 |
| 2 | 霍尔 |
| 3 | ADC采集 |
| 4 | 光电直读 |

### 3. 设置脉冲常数 (SetPulseConstant)

```json
{
    "method": "SetPulseConstant",
    "params": {
        "pulseConstant": 1
    }
}
```

| pulseConstant | 说明           |
|---------------|--------------|
| 1 | 1计量脉冲表示1升    |
| 2 | 1计量脉冲表示10升   |
| 3 | 1计量脉冲表示100升  |
| 4 | 1计量脉冲表示1000升 |
