# 物联网平台订阅服务定价设计指南

**文档版本**：v1.0  
**编制日期**：2026-03-24  
**适用范围**：LoRa 物联网 SaaS 平台  
**参考来源**：The Things Industries、Kerlink、Smart Parking 等竞品分析

---

## 一、定价设计原则

### 1.1 核心原则

| 原则 | 说明 | 竞品参考 |
|------|------|----------|
| **价值对齐** | 价格与客户获得的价值匹配 | TTI：设备数越多，价值越大 |
| **降低门槛** | 免费层吸引客户，降低试用成本 | TTI Discovery 免费层 |
| **灵活扩展** | 按需计费，支持从小规模到大规模 | TTI：1000设备后按需计费 |
| **透明可信** | 公开核心价格，减少销售阻力 | TTI：公开套餐对比 |
| **分层服务** | 不同客户不同服务级别 | SLA 分层、支持分层 |

### 1.2 定价目标

1. **市场渗透**：免费/低价吸引中小企业和开发者
2. **收入增长**：设备规模扩大带来收入增长
3. **增值变现**：场景服务、高级支持带来溢价
4. **客户留存**：长期订阅折扣、渠道分成激励

## 二、计费维度设计

### 2.1 主计费维度

| 维度 | 说明 | 推荐方式 | 参考 |
|------|------|----------|------|
| **设备数** | 网络中注册的终端设备数量 | 阶梯计费 | TTI 核心维度 |
| **网关数** | 接入网络的网关数量 | 套餐包含或单独计费 | TTI：Standard/Plus 无限 |
| **数据流量** | 上行/下行消息数量 | 按量计费（可选） | 辅助维度 |
| **存储容量** | 历史数据存储空间 | 按容量计费（可选） | 辅助维度 |

### 2.2 辅助计费维度

| 维度 | 说明 | 适用场景 |
|------|------|----------|
| **API 调用次数** | 外部接口调用频率 | 高频集成客户 |
| **用户数** | 平台账户数量 | 团队协作场景 |
| **应用数** | 创建的应用数量 | 多项目客户 |

### 2.3 计费周期

| 周期 | 折扣建议 | 说明 |
|------|----------|------|
| **月付** | 基准价格 | 灵活性高 |
| **年付** | 85-90 折 | 鼓励长期承诺 |
| **两年付** | 75-80 折 | 深度绑定 |

## 三、套餐分层设计

### 3.1 套餐结构

```
┌─────────────────────────────────────────────────────────┐
│                     套餐分层金字塔                        │
├─────────────────────────────────────────────────────────┤
│                    Enterprise（企业定制）                 │
│              私有部署 | 定制 SLA | 专属支持               │
├─────────────────────────────────────────────────────────┤
│                      Plus（高级版）                       │
│         场景服务 | 高级分析 | 优先支持 | 发票支持         │
├─────────────────────────────────────────────────────────┤
│                   Standard（标准版）                      │
│        基础服务 | 99.9% SLA | 数据查看 | 远程控制        │
├─────────────────────────────────────────────────────────┤
│                   Discovery（体验版）                     │
│              免费 | 功能试用 | 设备数限制                 │
└─────────────────────────────────────────────────────────┘
```

### 3.2 套餐详细设计

#### Discovery（体验版）- 免费

| 项目 | 配置 |
|------|------|
| **价格** | 免费 |
| **设备数** | ≤10 台 |
| **网关数** | ≤2 台 |
| **数据保留** | 7 天 |
| **SLA** | 无保证 |
| **支持** | 文档自助 |
| **适用场景** | 评估测试、小型 PoC、教育培训 |

**目标**：降低试用门槛，让客户体验核心功能，转化为付费用户。

## 四、设备数阶梯计费设计

### 4.1 阶梯定价模式

| 设备数区间 | 单价（参考） | 说明 |
|-----------|-------------|------|
| 1-100 | 套餐包含 | 基础费用 |
| 101-500 | ¥X/台/月 | 小规模 |
| 501-2000 | ¥Y/台/月（Y<X） | 中规模，单价降低 |
| 2001-10000 | ¥Z/台/月（Z<Y） | 大规模，单价更低 |
| 10001+ | 定制协议 | 超大规模，单独洽谈 |

### 4.2 计费示例

假设：
- Standard 套餐：¥1000/月（含 100 设备）
- 101-500 设备：¥5/台/月
- 501-2000 设备：¥3/台/月
- 2001+ 设备：¥2/台/月

**示例 1**：300 台设备
```
基础费用：¥1000
超出设备：(300-100) × ¥5 = ¥1000
月费用：¥2000
```

**示例 2**：1000 台设备
```
基础费用：¥1000
101-500 档：(500-100) × ¥5 = ¥2000
501-1000 档：(1000-500) × ¥3 = ¥1500
月费用：¥4500
```

## 五、SLA 分层设计

### 5.1 SLA 承诺

| 套餐 | 可用性承诺 | 年停机时间上限 | 服务积分 |
|------|-----------|---------------|----------|
| Discovery | 无 | - | 无 |
| Standard | 99.5% | 43.8 小时/年 | 10×(99.5%-实际) |
| Plus | 99.9% | 8.76 小时/年 | 10×(99.9%-实际) |
| Enterprise | 99.99% | 52.6 分钟/年 | 定制 |

### 5.2 响应时间承诺

| 问题级别 | Standard | Plus | Enterprise |
|----------|----------|------|------------|
| 关键组件宕机 | 4 小时 | 2 小时 | 30 分钟 |
| 关键组件降级 | 8 小时 | 4 小时 | 1 小时 |
| 非关键问题 | 24 小时 | 8 小时 | 4 小时 |

### 5.3 SLA 不适用场景

- 计划维护（提前 5 天通知）
- 紧急维护
- 客户端设备或网络问题
- 第三方服务故障
- 超出 API 速率限制
- 不可抗力

## 十、竞品真实定价参考（The Things Industries）

### 10.1 实际价格数据（2026-03-24 获取）

| 设备数 | Standard 套餐 | Plus 套餐 |
|--------|--------------|-----------|
| 500 | $230/月 | $360/月 |
| 1,500 | $420/月 | $540/月 |
| 5,000 | $840/月 | - |
| 12,500 | - | $1,800/月 |
| 50,000 | 联系销售 | 联系销售 |

### 10.2 价格规律分析

**Standard 套餐**：
- 500 设备：$230/月 = **$0.46/台/月**
- 1,500 设备：$420/月 = **$0.28/台/月**
- 5,000 设备：$840/月 = **$0.17/台/月**

**Plus 套餐**：
- 500 设备：$360/月 = **$0.72/台/月**
- 1,500 设备：$540/月 = **$0.36/台/月**
- 12,500 设备：$1,800/月 = **$0.14/台/月**

**规律**：
1. 设备数越多，单价越低（阶梯递减）
2. Plus 比 Standard 溢价约 50-60%
3. 大规模客户（5万+设备）需联系销售定制

### 10.3 推测定价公式

**Standard 套餐**（推测）：
- 基础费：约 $100/月（含少量设备）
- 设备单价：$0.15-0.35/台/月（阶梯递减）

**Plus 套餐**（推测）：
- 基础费：约 $200/月（含少量设备）
- 设备单价：$0.12-0.50/台/月（阶梯递减）
